Gen5 ST5adv

다양한 확장 프로그램을 관리하고 실행할 수 있는 플랫폼인 시작 매니저에 대한 설명 페이지입니다.

ST5adv

다른 프로그램을 관리하고 실행시킵니다.
인터넷을 통해 확장 프로그램 다운로드가 가능합니다.

프로그램 실행

시작 화면입니다.
일부 프로그램이 설치된 것을 볼 수 있습니다.
아이콘을 클릭하면 해당 프로그램이 실행됩니다.

testext5 확장 프로그램을 실행한 모습입니다.
ST5adv 설치 후 정상 동작하는지 확인을 위해
testext5을 설치하여 테스트하시기 바랍니다.

추가 기능

실행모드 설정 메뉴입니다.
아이콘 클릭 시 실행이 아니라 그 프로그램이 위치한 폴더를 열게 설정할 수 있습니다.

ST5adv 도구 -> 파일/폴더 선택기 메뉴입니다.
파일 또는 폴더를 선택하면 그 경로가 클립보드에 복사됩니다.

확장 프로그램 설치

웹 확장 도구 -> 다운로드 메뉴입니다.
인터넷 서버에 올라온 확장 프로그램의 설명을 볼 수 있습니다.
다운로드 버튼을 눌러 확장 프로그램 데이터를 다운로드합니다.

프로그램 실행파일이 위치한 폴더에 있는 설정 파일 텍스트입니다.
다른 항목은 바꿀 필요 없으나, export 항목은 직접 설정해야 합니다.
다운로드 데이터가 생성되는 폴더의 경로를 작성해주세요.

export로 설정한 폴더 안에 다운로드 데이터 zip 파일이 생성됩니다.
인증서(텍스트 파일)과 데이터 파일(*.숫자)을 전부 압축 해제하세요.
zip 안의 모든 파일을 동일한 폴더 내부에 빠짐없이 압축 해제해야 합니다.
그 후 로컬 확장 도구 -> 확장 가져오기 메뉴로 프로그램을 설치하세요.
로컬 확장 도구 -> 확장 관리 메뉴로 현재 설치된 프로그램의 정보를 확인할 수 있습니다.

프로그램 구조

st5adv~ 이름의 파일이 중요한 파일들입니다.
위 사진의 설정 파일도 st5adv_settings.txt 이름으로 존재합니다.
entrypoint 파일은 실행 프로그램이며, publickey는 공개키 파일입니다.

설치된 프로그램은 extension 폴더에 위치합니다.
이 설명서에서는 3개의 프로그램이 이미 설치된 상태입니다.

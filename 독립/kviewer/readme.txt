gen5 viewer [win & linux]
폴더/파일 보기와 관리를 할 수 있는 툴입니다.
KZIP5/simple div 작업도 간단하게 가능합니다.

kviewer5는 설정 파일에 주요 설정을 저장합니다.
프로그램 폴더 안 settings.txt 파일이 설정 파일입니다.
이 파일에 div 용량, 바로가기 경로, 시작 경로 등을 설정할 수 있습니다.
이 파일이 없을 경우, 자동으로 기본 설정이 적용됩니다.

<기본 UI>
처음 프로그램을 실행시키면, 상위 폴더로 가는 버튼, 현재 폴더 경로, 현재 폴더 안의 폴더와 파일들이 표시됩니다.
이때 폴더를 더블클릭하면 그 폴더로 이동하며, 파일을 더블클릭하면 그 파일이 열립니다.
오른쪽에는 설정에 따라 크기가 나타나는데, Viewsize 메뉴에서 Toggle로 켜고 끌 수 있습니다.

<Teleport 메뉴>
- 바로 가기 : 미리 설정해놓은 바로가기 폴더로 이동할 수 있습니다.
- 수동 지정 : 직접 폴더 경로를 입력해 이동할 수 있습니다.

<General 메뉴>
- 현재 폴더 열기 : 현재 보고 있는 폴더를 운영체제 기본 파일 탐색기로 엽니다.
- 선택 경로 복사 : 뷰어에서 선택한 폴더/파일의 경로를 클립보드에 복사합니다.
- 세부 정보 확인 : 선택 항목을 포함해서, 하위 폴더/파일 수와 크기, 수정 시각을 알려줍니다.

<Files 메뉴>
- kzip pack/unpack : stdlib5.kzip으로 폴더를 패키징하여 webp파일로 만들거나, 패키징된 파일을 해제합니다.
- div pack/unpack : div는 큰 파일 하나를 일정 용량대로 쪼개기만 합니다. 파일을 쪼개거나 합칠 수 있습니다.
- khash : 선택한 폴더/파일의 stdlib5.ksign-khash 값을 표시합니다. 값은 64바이트입니다.

<! viewsize 성능 경고 !>
viewsize 옵션을 켜 놓으면 창에 정보를 표시할 때마다 모든 폴더와 파일의 크기 정보를 가져옵니다.
하위 항목이 매우 많고 복잡한 폴더(운영체제가 있는 폴더 등)는 이 크기 구하기 과정에 시간이 걸립니다.
빠른 화면 표시를 위해서는 C:/ 같은 최상위 디렉토리에서는 viewsize 옵션을 꺼 놓는것을 추천합니다.

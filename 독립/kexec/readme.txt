gen5 Kexec [win only]
파일/프로그램 실행 모음집입니다.
폴더는 열 수 없으며, 파일은 그 파일이 들어있는 폴더를 작업 경로로 맞춰 열어줍니다. (실행파일 작업경로 관련)
파이썬 런타임 폴더로 구성되어 있으며, 안쪽 kexec.exe 파일이 실행파일, data599.txt 파일이 데이터 파일입니다.

데이터 파일에 실행하려는 모음집의 특성에 따라 제목을 지을 수 있습니다.
모든 파일/프로그램의 이름과 경로는 데이터파일에 작성합니다.
데이터 파일(data599.txt)은 stdlib5.kdb에 따라 유니코드 문자열로 작성합니다.
다음은 데이터 파일의 예시입니다.

title = "게임"
num = 2
0.path = "C:/bin/mojang/minecraft.exe"
0.name = "마인크래프트"
1.path = "C:\program\steam\ksp\ksp.exe"
1.name = "KSP"

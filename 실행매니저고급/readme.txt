ST5adv : 5세대 프로그램 매니저 고급 [win & linux]
다양한 ST5adv 확장 프로그램을 설치, 관리, 다운로드, 실행할 수 있는 플렛폼입니다.
전자서명과 웹 연결을 통해 더 안전하거 편리하게 프로그램을 관리할 수 있습니다.

기본적으로 testext5와 testlib5 확장을 제공합니다.
ST5adv 설치 후 두 확장을 다운로드 받아 테스트해보세요.
또한 ST5adv 프로그램 자체도 하나의 확장 프로그램으로써 동작할 수 있는 구조입니다.
리눅스 상에서 Wine을 사용할 때 윈도우 전용 확장 프로그램 실행용으로 사용하실 수 있습니다.

<초기 설정>
프로그램 폴더 안 st5adv_settings.txt 파일을 조작해 초기 설정을 해 주어야 합니다.
먼저 다운로드 폴더를 설정해야 합니다. export = "경로명" 으로 바꿔주세요.
확장 배포를 하려는 경우, 개인키 파일 경로도 설정해야 합니다. pvk = "경로명" 으로 바꿔주세요.
나머지 항목은 바꿀 필요 없이, 잘못된 부분이 있는지만 체크하세요.

<확장 실행>
한 화면당 6개씩 확장 프로그램을 보여줍니다. 각 화면 아래에 이름이 표시됩니다.
화면 버튼을 클릭해 해당 확장을 실행하거나 위치한 폴더를 열 수 있습니다.
좌/우의 버튼으로 이전/다음 확장 프로그램을 볼 수 있습니다.
상단의 "실행모드 설정" 메뉴로 "파일 실행" / "폴더 열기" 모드를 바꿀 수 있습니다.

<ST5adv 도구>
현재 ST5adv 폴더를 열거나, 파일 / 폴더 선택 도구를 실행할 수 있습니다.
선택 도구는 리눅스 형식이며, 따옴표 없이 클립보드에 복사됩니다.

<웹 다운로드>
ST5adv 확장들에 대한 설명을 볼 수 있습니다.
또한 현재 웹 저장소에 올라간 확장들의 목록과 설명을 볼 수 있습니다.
다운로드 할 경우, 확장 설치에 필요한 *.0, *.txt 파일이 압축된 파일이 받아집니다.
이 파일을 압축해제하여 확장 설치에 사용하면 됩니다.
표시되는 항목은 이름_운영체제 + 버전 숫자 형식입니다. 각 운영체제별로 맞는 항목만 표시됩니다.

<확장 관리>
설치한 확장 프로그램의 정보를 볼 수 있습니다.
확장 프로그램 삭제도 가능합니다.
이 경우 메인 화면으로 돌아갈 때 "back to Main" 버튼을 눌러야 정상 동작합니다.

<확장 설치>
로컬 컴퓨터의 위치한 *.0와 *.txt 파일을 입력하여 해당 확장을 설치할 수 있습니다.
*.숫자 형식의 파일은 시리즈 전체가 있어야 합니다. (웹 다운로드 받은 압축 파일 내부)
전자서명 모드를 꺼 놓았다면 *.txt 선택은 취소하셔도 됩니다.

<확장 배포>
확장 프로그램의 이름, 전자서명 개인키 비밀번호를 입력하고 내보낼 폴더를 선택하여
*.0 ~ 파일과 *.txt 서명 파일을 생성합니다. 이 파일들을 압축 / 비압축으로 배포하면 됩니다.
패키지 이름은 자유롭게 지어도 되나, 실제 프로그램이 설치될 때는 폴더 자체가 그대로 설치되기에
폴더 이름은 겹치지 않게 확장 프로그램 이름을 지어야 합니다.
전자서명은 KHash5와 호환되며, 전자서명 없이 배포하려면 먼저 아무 서명이나 만든 후
*.txt 서명 파일을 제외하고 배포하면 됩니다.

확장 패키지 폴더는 다음 구조로 이뤄집니다.
/%pkg name%
    /code
        /*.*
    /st5adv_entrypoint.exe
    /st5adv_extensionicon.png
    /st5adv_versioninfo.txt
코드 폴더와 버전 정보 텍스트, 아이콘 사진은 없어도 됩니다.
버전 정보가 없다면 Unknown으로 표시되며, 아이콘 사진이 없다면 기본 사진으로 표시됩니다.
이 경우 프로그램 실행은 가능합니다. 다만 entrypoint 실행파일이 없다면 파일 실행이 불가능합니다.

웹 배포는 kweb을 통해 이루어집니다.
확장 프로그램 내보내기로 *.숫자, *.txt 서명 파일이 생성됩니다.
이 파일을 모두 압축하여 *.zip과 같이 단일 파일로 만듭니다.
이 단일 파일을 KViewer5 등의 프로그램으로 div pack으로 나눠 *.숫자 형식의 파일로 다시 쪼갭니다.
stdlib5.kweb에 호환되도록, html 상에 id 정보와 url 정보를 잘 입력시키고
같은 디렉토리 상에 모든 *.숫자 파일을 올립니다.
kweb 라이브러리에 원본 압축파일 이름, 청ㅋ크 개수를 입력하면 다운로드 받을 수 있습니다.

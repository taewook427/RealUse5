extension.kpwm5 [win & linux] : 비밀번호 문서 관리기.

<PWKF, Info>
비밀번호와 키파일을 재설정하거나 현재 비밀번호를 볼 수 있습니다.

<Sort>
데이터를 이름순 정렬합니다.

<Lock>
데이터 수정가능 여부를 설정합니다.

<Next>
다음 라인으로 이동하고, 라인의 끝이라면 다음 페이지로 이동합니다.

<Save>
변경사항을 저장합니다. 데이터 수정, 추가, 삭제 모두 Save를 눌러야 저장됩니다.

<Import, Export>
데이터를 평문 텍스트로 내보내거나 가져옵니다.
텍스트는 kdoc 마크업 형식입니다.

<AddPage, DelPage>
페이지를 추가하거나 삭제합니다.
페이지는 같은 주제의 데이터를 가진 여러 개의 라인으로 이루어져 있습니다.

<AddLine, DelLine>
라인을 추가하거나 삭제합니다.
라인은 (title, content)로 이루어져 있으며, 항목 하나에 대한 정보를 나타냅니다.

<로그인>
데이터 청크를 선택하지 않고 로그인하면 새 저장소 만들기 모드로 실행됩니다.
데이터청크(webp) 파일을 직접 선택하거나 경로가 적힌 텍스트(txt) 파일을 선택하여 로그인합니다.

비밀번호나 기밀 텍스트 내용을 저장할 수 있습니다.
페이지와 라인 단위로 주제별로 항목을 나눌 수 있으며,
좌측 하단의 특수버튼으로 검색 기능도 지원합니다.

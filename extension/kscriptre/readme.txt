extension.kscriptre [win & linux] : kscript CLI 런타임.

<실행 옵션 설정>
safemem : 안전한 메모리 접근
runone : 저속 모드
errhlt : 예외 시 멈춤없이 계속 진행
maxstk : 최대 스택 크기 (음수면 기본값 사용)
chksign : 신뢰할 수 있게 서명된 실행파일만 로드

<외부함수 옵션>
N.name, N.code, N.using로 설정하는 외부함수 옵션은
해당 외부함수 라이브러리를 사용할지를 결정합니다.

kscript의 go 버전 런타임입니다.
_ST5_DATA/ 폴더에 실행 바이너리를 넣으면 선택 후 실행합니다.

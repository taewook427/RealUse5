extension.kcmd5 [win & linux] : 쉘 명령어 실행기.

<./_ST5_DATA/>
패키지 폴더의 데이터 폴더에 쉘 명령어들이 위치.
데이터 폴더에 KDB 텍스트 형태로 실행할 명령어들이 들어있다.

<쉘 명령어 저장 형식>
텍스트 파일은 여러 개 존재할 수 있으며, 각 명령어는 순차적으로 실행됨.
XXX.txt
    0.name = "출력"; 0.0 = "echo Hello!"
    1.name = "이동"; 1.0 = "move ~ ~"; 1.1 = "timeout 1"
YYY.txt
    ...

선택된 쉘 명령어를 실행하여 프로그램이나 파일 관리를 도와줍니다.
윈도우에서는 항상 관리자 권한으로 실행되며, 리눅스에서는 현재 유저 권한으로 실행됩니다.

extension.convs [win & linux] : 사진/영상/문서 변환기.

<사진>
img resize / img conv로 다음 작업 가능.
기능 1 : 사진 해상도와 형식을 바꿉니다.
기능 2 : 여러 사진을 하나의 사진으로 병합합니다.
기능 3 : 사진을 회전/대칭합니다.
기능 4 : 흑백화, 색반전, 색강조 등 특수효과를 입힙니다.

<동영상>
vid resize / vid conv로 다음 작업 가능.
기능 1 : 동영상 재인코딩 (해상도, 코덱, 비트레이트 등 설정 가능)
기능 2 : 동영상 자르기 (mp4-mp4)
기능 3 : 소리 추출 / 변환 (mp4-mp3, mp3-mp4)
기능 4 : 움짤 추출 / 변환 (mp4-gif, gif-mp4)

<PDF>
pdf conv로 다음 작업 가능. (내용 수정은 불가, 더 자세한 작업은 kscript.pdf 필요)
기능 1 : 페이지 순서 바꾸기
기능 2 : 페이지 회전하기

<대량변경>
ext conv로 다수의 파일을 일괄 변환하거나 병합 가능.
기능 1 : 사진을 jpg/png/webp 형식으로 일괄 변환.
기능 2 : html 문서를 pdf 문서로 일괄 변환.
기능 3 : 여러 개의 사진/pdf를 하나의 pdf 문서로 병합.
기능 4 : 여러 개의 동영상을 하나의 동영상으로 병합.
(동영상들이 같은 형식으로 인코딩 되었다면 cp 옵션으로 재인코딩 없이 빠르게 진행 가능.)

사진과 동영상, pdf 파일에 일상적인 편집 작업을 할 수 있습니다.
웹 서비스를 이용해야 하거나 따로 전용 프로그램을 설치해야 하는 작업을 가볍게 할 수 있습니다.
결과물은 데이터 폴더 (_ST5_DATA)에 저장됩니다.

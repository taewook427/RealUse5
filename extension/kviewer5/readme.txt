extension.kviewer5 [win & linux] : 기능 추가된 파일 탐색기.

<settings.txt>
설정 파일로 바로가기 항목을 지정합니다.
auto : 이 수치보다 폴더 깊이가 높아야 폴더 크기 보기가 켜집니다.
size : 텍스트/바이트 보기에서 너무 큰 파일을 불러오는 것을 막기 위한 제한 수치입니다.
N.name, N.path : 바로가기 이름과 경로를 설정합니다. path는 절대 경로여야 합니다.

몇 가지 기능을 추가한 파일 탐색기입니다.
기본 탐색기가 부족하거나 멈춰 버린다면 사용하세요.

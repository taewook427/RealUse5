extension.kmacro [win & linux] : 자동화 작업 보조기.

<capture>
특정 키보드, 버튼 클릭으로 화면 캡쳐 가능
Lock : 항상 화면에 보이는 옵션
Cut : 사각 자르기 캡쳐 여부

<downloader>
웹 페이지나 유튜브 영상을 다운로드.
web : 웹 페이지를 pdf나 사진 형태로 다운받거나 페이지 소스(html js css)를 가져오기.
ytube : 유튜브 영상 다운로드 (기본값 최고화질, 옵션조정 가능)
ytube info : 다운로더 래퍼(yt-dlp)의 URL에 대한 자세한 다운로드 옵션 출력.

<kscript>
kscript (매크로 제작용 언어) 텍스트 소스코드 또는 컴파일된 바이너리를 받아서 실행.
errhlt가 켜지면 일반오류 발생시에도 멈추고,
커스텀 스택 값을 양수로 입력하면 최대 스택 크기를 조작할 수 있습니다.
Clear : 텍스트 로그 지우기
Enter : 텍스트 한줄 입력 (입력창을 선택하고 키보드 엔터를 눌러도 가능)
Run : 실행 여부, 이것이 활성화되어야 프로그램이 실행됨

!!! 리눅스용을 제공하지만, 일부 기능이 작동하지 않을 수 있습니다 !!!
화면 캡쳐, 웹/유튜브 다운로드, kscript 실행 등 자동화 작업을 실행 보조합니다.
결과물은 데이터 폴더 (_ST5_DATA)에 저장됩니다.

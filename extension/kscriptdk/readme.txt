extension.kscriptdk [win & linux] : kscript 개발도구.

<compiler>
소스코드를 수정하거나, 바이너리로 빌드할 수 있습니다.
버튼들은 최적화 옵션, 중간단계 표시 옵션, 각 외부함수 기능 사용 여부입니다.
save : 화면에 보이는 텍스트(수정된 소스코드)를 원본 경로에 저장합니다.
build : 프로그램 정보, 아이콘, 서명 등을 사용해서 컴파일하고 빌드합니다.
run : 빌드한 프로그램을 바로 디버그 모드로 실행합니다.
컴파일 결과물은 데이터 폴더(_ST5_DATA)에 있습니다.

<debugger>
프로그램을 실행하며 모든 스택과 레지스터 값을 볼 수 있습니다.
텍스트 콘솔 기능은 kmacro와 동일하며, 오른쪽에 스택과 레지스터 부분이 추가되었습니다.
프로그램이 너무 큰 값을 다루거나 실행 사이클이 많다면 사용하지 마십시오.

kscript 소스코드를 바이너리로 빌드하거나 디버그 모드로 실행시킬 수 있습니다.
디버그 모드는 모든 스택과 레지스터가 보이기 때문에 한 명령어씩만 실행되며,
작동 테스트용으로만 사용하는 것을 추천합니다.

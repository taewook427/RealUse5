extension.kmap [win & linux] : 3D 모델/지도 표시기.

<키보드 조작>
WASD, 방향키 : 수평 이동
Space, Shift : 수직 이동
-, + : 시야각 조작
[, ] : 배율 조작 (배율이 높을수록 빠르게 움직이고 물체가 크게 보여 넓은 맵에 좋습니다)
L : 라벨 키거나 끄기

<마우스 조작>
드래그 : 시야 움직임
스크롤 : 배율 조작

! 모델 파일 형식 !
kdoc의 마크업 형식을 사용하며 페이지로 점과 선, 설정을 구분하고
각 라인이 하나의 객체나 설정을 의미합니다.
<구분>
{객체명/설정명} (데이터)

<node>
{고유한 객체명} (lbl=""; color=""; r=0; x=0; y=0; z=0)
# 라벨, 색, 크기와 위치(필수 요소)를 설정합니다.

<line>
{고유한 객체명} (lbl=""; color=""; r=0; start="시작노드 객체명"; end="끝노드 객체명")
# 라벨, 색, 크기와 양쪽 노드(필수 요소)를 설정합니다.

<config>
{baseline} (0) # +-N 길이의 xyz축을 추가합니다. 색 : (x black, y blue, z red)
{viewlbl} (t) # 노드와 라인의 라벨 표시를 설정합니다.
{viewcord} (t) # 라벨에 좌표/선 길이를 표시할지 설정합니다.
{pos} (0, 0, 0) # 시작 좌표를 설정합니다.
{mul} (1.0) # 시작 배율을 설정합니다.
{fov} (90) # 시작 시야각을 설정합니다.

점과 선으로 이루어진 간단한 삼차원 모델을 보여줍니다.
delta-V map 등 게임 지도 보조용으로 개발되었습니다.

extension.ksync [win & linux] : 클러스터 동기화 도구.

<Cluster Sync>
src(원본)에서 dst(사본)폴더로 동기화, 원본은 데이터 불변.
서로 같은 내용을 가지도록 폴더/파일을 삭제하거나 생성, 복사합니다.
옵션에 따라 파일이 같은지 판단하는 기준이 달라집니다.
name : 이름이 같다면 같은 파일
size : 크기가 같다면 같은 파일
time : 수정시각이 같다면 같은 파일
data : 바이트 내용이 같다면 같은 파일

<Cluster Patch>
src(원본)에서 dst(사본)폴더로 동기화, 원본은 데이터 불변.
Cluster Sync는 파일 기반으로 관리하기에, 파일이 다르다면 해당 파일을 전부 복사했습니다.
옵션에 따라서 파일이 단 1 바이트만 달라도 전체 데이터를 복사해야 합니다.
더 효율적인 복사를 위해, patch는 파일을 1MiB 크기로 나눠 다른 지점이 존재하는 부분만 덮어씁니다.

<Change Time>
tgt폴더와 하위 요소의 수정 시각 변조.
now : 현재 시각으로 변조.
rand : 현재 시각에서 3년 전까지의 기간 사이 무작위 날자로 변조.
preset : yyyy.mm.dd 형식으로 설정할 날자를 입력받아 변조.

폴더의 수정 시각을 바꾸거나 다른 폴더와 내용물을 동기화할 수 있습니다.

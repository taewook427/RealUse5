independent.khash5 [win & linux] : 전자서명 도구.

<mode 0>
프로그램 종료.

<mode 1>
파일/폴더의 khash 해시값 구하기.
모드 진입 시 폴더/파일 선택창이 나옵니다. "1"을 입력해 파일 경로를, "2"를 입력해 폴더 경로를 입력합니다.
경로 입력 시 전체 경로를 직접 입력하거나 숫자 번호를 입력하세요.
폴더 해싱 모드라면 "3 f", 파일 해싱 모드라면 "3 t"를 입력해 파일 모드를 설정하세요.
모두 입력 후 "submit"를 입력하면 해싱 결과가 표시됩니다.

<mode 2>
파일/폴더 서명하기.
모드 1과 비슷하게 폴더/파일 경로를 설정하고 폴더 대상인지 파일 대상인지 설정합니다.
"4"를 입력해 개인키 파일을 설정합니다. "direct"로 개인키 파일 경로를 직접 입력하거나,
"comm ~"로 kcom 통신으로 받아올 수 있습니다.
모두 입력 후 "submit"를 입력하면 대상 패키지가 위치한 곳에 sign.txt가 생성됩니다.

<mode 3>
파일/폴더 서명 검증하기.
모드 1과 비슷하게 폴더/파일 경로를 설정하고 폴더 대상인지 파일 대상인지 설정합니다.
"4"를 입력해 서명 파일을 설정합니다. 서명 파일은 txt 파일만 인식됩니다.
모두 입력 후 "submit"를 입력하면 올바른 서명인지 결과가 표시됩니다.

<mode 4>
새 서명 키 파일 생성.
"1 %이름%"을 입력해 키 파일의 이름을 설정합니다.
"2 %숫자%"를 입력해 RSA 강도를 설정합니다. 1024 ~ 16384의 2의 거듭제곱이여야 합니다.
모두 입력 후 "submit"를 입력하면 private.txt(개인키)와 public.txt(공개키)가 생성됩니다.
공개키는 공개하고 개인키는 평문 형태라 매우 취약하니 kboom등 안전한 저장소에 보관하세요.

RSA 서명 키를 생성하거나 패키지에 서명할 수 있는 도구입니다.
프로그램 메인 화면에서 0~4를 입력해 각 모드를 실행합니다.

independent.kboom5 [win & linux] : 키 파일 관리 도구.

<password>
비밀번호와 키 파일, 힌트를 설정하는 창입니다.
위쪽의 두 입력창은 키 파일을 설정합니다. 위쪽에 있는 버튼으로 직접 파일 탐색기를 작동시키거나,
아래쪽에 kcom 통신 주소를 입력하고 키 파일을 받아올 수 있습니다.
바로 밑쪽엔 비밀번호 힌트가 표시됩니다. 비밀번호 설정 모드에서는 힌트 문자열을 직접 입력하게 됩니다.
최하단에는 비밀번호 입력란이 있습니다. 모두 올바르게 입력했다면 Go 버튼을 누르세요.

<mainpage>
메인화면엔 저장소 폴더와 각 폴더에 저장된 파일들이 보입니다.
텍스트 파일은 오른쪽 박스에서 내용을 확인할 수 있습니다.

<PWreset>
비밀번호, 키 파일, 비밀번호 힌트를 재설정합니다.

<boom!>
저장된 모든 파일을 지우고 저장소를 초기화합니다.

<Import>
현재 저장소에 파일들을 가져옵니다.
파일 가져오기 이후에 프로그램을 종료하고 다시 실행하는 것을 권장합니다.

<Export>
현재 저장소의 모든 파일을 지정된 폴더에 복호화합니다.

<Delete>
선택된 키 파일을 제거합니다.

<View>
현재 저장소 하에 있는 사진 파일을 메모리에 불러옵니다.
좌하단 버튼으로 사진을 볼 수 있습니다.

<Send>
kcom 통신으로 키 파일을 보냅니다. 드라이브에 평문 파일이 남지 않기 때문에 더 안전합니다.

<Edit>
텍스트 파일을 우측 박스에서 내용을 수정할 수 있게 합니다.
수정 후 새로고침을 눌러야 저장됩니다.

키 파일이나 서명 개인키 등을 관리할 수 있는 프로그램입니다.
키 파일은 로컬 저장소에 존재하며 더 안전한 비밀번호 역할을 하는 파일입니다.
하지만 USB 등에 들어있다면 공격자 탈취 시 그대로 노출될 위험이 있습니다.
kboom5는 키 파일을 암호화하여 저장하고 kcom 통신으로 키 파일을 전송하여 더 높은 보안성을 가지도록 도와줍니다.

구동에 폴더가 필요합니다. 프로그램 파일은 로컬 저장소나 USB 등에 위치하며,
기본 저장소 폴더 외에도 다른 이름의 폴더를 직접 만들거나 수정할 수 있습니다.

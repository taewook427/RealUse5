independent.installer [win & linux] : 프로그램 다운로더.

<dev>
unpack [path] : 다운로드한 패키지를 언패킹합니다.
download [url] : 모든 패키지 목록을 보고 다운로드할 항목을 선택합니다.
-1로 종료하며, url 미지정시 내장 url이 사용됩니다.

<user>
폴더 경로를 설정하고 다운로드할 항목을 선택하면 자동으로 다운로드 후 설치가 진행됩니다.
바탕화면에 그대로 풀리거나, 시작 프로그램 폴더에 풀리거나, 홈 폴더에 풀린 후 바로가기가 생성됩니다.

수동 다운로더(go)와 자동 다운로더(py)입니다.
자동 다운로더는 윈도우만 지원합니다.

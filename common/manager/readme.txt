common.manager [win & linux] : 사용자용 패키지 매니저.

<Status>
Starter5의 설정 정보 보기.
미리 설정된 경로들, 링크 주소들, 설정과 신뢰하는 공개키를 볼 수 있습니다.
개발자 모드 : 참이라면 더 다양한 패키지를 설치할 수 있습니다.
신뢰하는 공개키 : 이곳의 공개키는 안전한 발급기관에서 생성되었다고 간주됩니다.

<Installed>
로컬 extension/common에 설치된 패키지 보기.
Delete 버튼으로 패키지를 제거할 수 있습니다.

<All Packages>
웹 상에서 패키지 정보를 받아와 현재 제공되는 모든 패키지 보기.
Install 버튼으로 패키지를 다운로드하고 설치할 수 있습니다.

<Update Queue>
업데이트 가능한 패키지 보기.
웹 정보에 더 최신 버전의 패키지가 있다면 표시됩니다.
Update 버튼으로 패키지 업데이트를 할 수 있습니다.

<WebPage>
웹 설명서 페이지 열기.

<새로고침>
정보 다시 불러오기.
로컬 데이터와 웹 페이지에 다시 접속해 정보를 갱신합니다.
패키지 관리 후 새로고침을 할 것을 권장합니다.

<특수버튼>
좌하단 특수버튼으로 내용을 검색할 수 있습니다.

! 패키지 관리 규칙 !
모든 패키지는 하나의 이름공간 안에 각자의 이름(영문 10자 내외)으로 구분됩니다.
이름은 겹칠 수 없으며, 패키지 업데이트 시 _ST5_DATA/ 폴더는 보존됩니다.

로컬 실행 매니저에 설치된 프로그램들을 관리하고, 웹에서 추가 패키지를 받아올 수 있습니다.
사용자라면 config/sign 설정 파일을 저수준으로 제어하지 말고 이 프로그램을 사용하여 패키지를 관리하십시오.
